# 📖 Overview

**Magiil Mart** is a cross-platform e-commerce mobile application developed using **Flutter** and **Supabase**. It provides customers with a smooth shopping experience while enabling administrators to efficiently manage products, inventory, and customer orders.

The application features secure authentication, role-based access control, real-time database integration, online payment support, and location-based delivery address selection.

---

# ✨ Features

## 👤 Authentication

* Secure User Registration & Login
* Email Authentication
* Session Management
* Role-Based Access (Admin & Customer)

### 🛍️ Customer Features

* Browse products by category
* Search for products
* View detailed product information
* Add products to cart
* Update cart quantity
* Remove items from cart
* Checkout with Cash on Delivery (COD)
* Razorpay payment integration
* View order history
* Manage profile information

### 👨‍💼 Admin Features

* Admin Dashboard
* Add new products
* Edit product details
* Delete products
* Manage inventory
* View customer orders
* Update order status
* Upload product images

### 📍 Location Services

* Current location detection
* Delivery address selection
* OpenStreetMap integration
* Reverse geocoding for readable addresses

### 🖼️ Image Management

* Product image upload
* Supabase Storage integration
* Cached image loading for better performance

### 🔐 Security

* Supabase Authentication
* Role-Based Authorization
* Protected Admin Routes
* Secure Database Access
